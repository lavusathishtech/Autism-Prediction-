# Autism-Prediction-
This project focuses on predicting Autism Spectrum Disorder (ASD) using machine learning techniques. The goal is to analyze input features and build a model that can classify whether a person is likely to have autism

---

## 🚀 Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature selection
- Machine Learning model training
- Model evaluation (Accuracy, Precision, Recall)
- Prediction system

---

## 🛠️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib & Seaborn
- Scikit-learn
- Jupyter Notebook
